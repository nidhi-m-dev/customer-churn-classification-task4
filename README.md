# Customer Churn Classification - Task 4

## Classification Models, Evaluation Metrics & Handling Imbalanced Data

This project focuses on predicting customer churn using machine learning classification techniques.

## Objective
To build and evaluate a classification model for predicting whether a customer will leave the service or stay.

## Dataset
Telco Customer Churn Dataset

## Tools Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Models Used
- Logistic Regression
- Balanced Logistic Regression

## Evaluation Metrics
- Confusion Matrix
- Precision
- Recall
- F1-Score
- ROC Curve
- AUC Score

## Key Concepts Applied
- Data Cleaning
- Label Encoding
- Feature Scaling
- Classification Modeling
- Handling Imbalanced Data

## Conclusion
Logistic Regression performed effectively for churn prediction, and handling class imbalance improved minority class detection.
